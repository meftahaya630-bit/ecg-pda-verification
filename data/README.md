# Data Directory

This directory contains the PhysioNet ECG Eye-Tracking dataset.

## Download Instructions

1. Visit: https://physionet.org/content/eye-tracking-ecg/1.0.0/
2. Download the dataset
3. Place files in `raw/` directory

**Contents:**
- 63 participants across 6 expertise levels
- 630 ECG interpretation sessions
- 10 different cardiac pathologies
- Eye-tracking data with fixations and AOI information

**Note:** Raw data files are not committed to git (see `.gitignore`)
